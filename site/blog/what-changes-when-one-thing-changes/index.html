What changes when you change just one thing?

Why changing one detail can tell us more about a model than a score can.

Many tests ask whether a model produced the right answer. I ask a different question: when one important piece of evidence changes, does the answer change for a sensible reason?

This essay is still being written. It will use examples from my work on AI tools for students learning to code to show why changing one detail at a time can be so revealing.
